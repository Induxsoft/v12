# V12.
El paquete V12 (Base de plataforma y plantillas Web) es el módulo fundamental que provee las plantillas de sitio web y programas base de la plataforma V12, sirviendo como la infraestructura sobre la cual se construyen y operan los demás paquetes del ERP. Su propósito es ofrecer un entorno estandarizado, flexible y seguro para el desarrollo y despliegue de aplicaciones empresariales.
